ℹ️ About Not Mind

Features

Professional mind mapping and notes application with VS Code-like file system integration.

• Open any folder from your computer

• Notes and mind maps saved as JSON files

• Auto-save support

• Multiple theme options

• Rich text editor with grid alignment

• 14 mind map structures with curved arrows

• Insert images, hyperlinks, notes, sketches, PDFs into mind map nodes

• Enhanced sketch pad: pen, marker, shapes, text, undo

• Hyperlinks in notes (Ctrl+K)

• Sketch directly in notes

• New page support in notes
